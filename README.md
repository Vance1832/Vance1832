## About Me

I'm **Khant Zayar**, a Computer Science student at Rangsit University working toward a career in cybersecurity. The main focus is learning how systems get attacked, how defenders think, and how to build a security mindset from the ground up — SOC analysis, network security, and penetration testing.

Software development is a parallel track. I build things with Next.js and Django because knowing how applications are built makes you sharper at finding where they break.

- Studying **SOC analysis**, network security, and penetration testing
- Practicing on **TryHackMe** — working through CTF challenges and labs
- Building **[Flowbit](https://github.com/Vance1832/flowbit)** — a fintech ledger and settlement platform
- Also working in computer vision and game development on the side
- Reach me at **khant.zayar.dev@gmail.com**
